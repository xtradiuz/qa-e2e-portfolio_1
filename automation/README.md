# UI Automation Testing – Playwright

This module contains automated UI test coverage for the SauceDemo QA Portfolio, implemented using Playwright and JavaScript.

The goal of this automation suite is to validate critical user workflows and ensure application stability through repeatable regression testing.

---

## 🧰 Tools & Technologies

- Playwright
- JavaScript (Node.js)
- GitHub Actions (CI/CD integration)

---

## 🧪 Test Coverage

The automation suite covers key end-to-end user scenarios:

- Login validation (valid, invalid, locked-out user)
- Product page verification
- Add and remove items from cart
- Cart state validation
- Checkout process validation (required fields, completion flow)
- Logout functionality
