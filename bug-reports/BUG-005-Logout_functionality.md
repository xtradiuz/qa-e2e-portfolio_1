# BUG-005: Logout Action Does Not Provide Confirmation or Feedback

## Severity
Low

## Priority
Low

## Environment
- Browser: Chrome
- OS: Windows 11
- Application: SauceDemo

## Steps to Reproduce
1. Log in.
2. Click Logout.

## Expected Result
User should receive confirmation (e.g., message or notification).

## Actual Result
User is redirected to login page with no confirmation.

## Issue
Lack of logout confirmation may confuse users.

## Status
Open

## Notes
This is a UX improvement.
